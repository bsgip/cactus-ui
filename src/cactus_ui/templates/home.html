Quick Start Guide

Run your CSIP-Australia client against our testing harness in three steps.

1. Download Your Client Certificate

CSIP-Australia (based on IEEE 2030.5) uses mutual TLS for authentication. Download your certificate from the Certificate page and install it in your client.

Note: Certificates expire. Revisit the page to refresh and download a new one when needed.

2. Enable your domain for notifications (Optional)

IEEE 2030.5 allows the registration of arbitrary URIs to receive HTTP notifications. This tool will allow you to register a single authorised domain that can be used to receive these notifications. You can configure this domain from the Domain page

Note: Attempting to create a subscription using a notification URI outside of this configured domain will raise an error.

3. Review Test Procedures

Go to the Test Procedures page for the full list of available procedures with descriptions. These follow the DEIP Interoperability Steering Committee – Common Smart Inverter Profile (Australia) – Test Procedures, version 1.2.

4. Use the Runs Page

Go to the Runs page to manage everything test-related:

  • Trigger a new test run by selecting a procedure from the dropdown and clicking Start Run. You'll be given a test URL to point your client at.
    • Note: It can take up to 30 seconds for your test environment to be ready.
  • Once the client has completed the procedure steps, click Finalise to compile results. A ZIP file will be returned.
  • Download past results at any time from the same page.
